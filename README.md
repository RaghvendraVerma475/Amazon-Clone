# Amazon Clone

A simple front-end clone of the Amazon homepage created using basic web technologies.  
This project is made for learning and practicing HTML and CSS.


## Project Description

This Amazon Clone is a static website that looks similar to the Amazon homepage.
It focuses on layout design, and styling.
No backend or real shopping functionality is included.
